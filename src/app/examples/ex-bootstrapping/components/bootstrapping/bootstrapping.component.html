Bootstrapping

When you press F5, you see pacman eating dots.

This is a gif, located in the assets folder. You see it because Angular has not yet been loaded and this image is used as a placeholder in the <app-root> element of index.html. The Angular application will replace it's contents with the component bootstrapped in the root module. That module is defined in main.ts. To make it a bit slower, I have added a timeout of 1 second. After the app is loaded, you get this (but a little bit different).
