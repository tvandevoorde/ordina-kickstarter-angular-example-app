NgIf: conditional display

Will show the element when the value is true.

component code
isTitleVisible = false;
title = 'Hello World!';

toggleTitleVisibility(): void {
this.isTitleVisible = !this.isTitleVisible;
}

component template
<div *ngIf="isTitleVisible">{{title}}</div>
<button (click)="toggleTitleVisibility()">Show Title</button>
{{isTitleVisible}}
output
{{title}}
{{isTitleVisible}}
note

You can also use ng-container instead of the div or you could use
<ng-template [ngIf]="...">...</ng-template>

NgIf with else: conditional display

Will show the element when the value is true, another when the value is false.

component code
isTitleVisible = false;
title = 'Hello World!';

toggleTitleVisibility(): void {
this.isTitleVisible = !this.isTitleVisible;
}

component template
<div *ngIf="isTitleVisible; else elseBlock">{{title}}</div>
<ng-template #elseBlock>
  <div>There is no title!</div>
</ng-template>
<button (click)="toggleTitleVisibility()">Toggle</button>
{{isTitleVisible}}

NgFor: for-loop

Loops over a list of items.

component code
numbers = ['one', 'two', 'three'];
output
{{ i }} {{ num }}
component template
<span *ngFor="let num of numbers; let i=index;">
{{ i }} {{ num }}
</span>
note

You can also use ng-container instead of the div or you could use
<ng-template [ngFor]="...">...</ng-template>

NgSwitch: switch statement

specifies an expression to match against

component code
rolledNumber: number;
roll(): void &"#123;
  this.rolledNumber = Math.floor(Math.random() * 6) + 1;
}
output

{{rolledNumber}}
Too bad we're not playing cards! Bad luck! Halfway there! Better luck next time! Very good! Jackpot!
component template
<button (click)="roll()">Roll</button>
{{rolledNumber}}
<div [ngSwitch]="rolledNumber">
  <span *ngSwitchCase="1">Too bad we're not playing cards!</span>
  <span *ngSwitchCase="2">Bad luck!</span>
  <span *ngSwitchCase="3">Halfway there!</span>
  <span *ngSwitchCase="4">Better luck next time!</span>
  <span *ngSwitchCase="5">Very good!</span>
  <span *ngSwitchCase="6">Jackpot!</span>
</div>

NgContainer, NgTemplate and NgTemplateOutlet

Using NgContainer, we do not need the div for NgSwitch.

We can define templates using NgTemplate and use them using NgTemplateOutlet.

component code
rolledNumber: number;
roll(): void &"#123;
  this.rolledNumber = Math.floor(Math.random() * 6) + 1;
}
output

{{rolledNumber}} Too bad we're not playing cards! Bad luck! Halfway there! Better luck next time! Very good! Jackpot!
component template
<button (click)="roll()">Roll</button>
{{rolledNumber}}
<ng-container [ngSwitch]="rolledNumber">
<ng-container *ngSwitchCase="1"><ng-container *ngTemplateOutlet="roll1"></ng-container></ng-container>
<ng-container *ngSwitchCase="2"><ng-container *ngTemplateOutlet="roll2"></ng-container></ng-container>
<ng-container *ngSwitchCase="3"><ng-container *ngTemplateOutlet="roll3"></ng-container></ng-container>
<ng-container *ngSwitchCase="4"><ng-container *ngTemplateOutlet="roll4"></ng-container></ng-container>
<ng-container *ngSwitchCase="5"><ng-container *ngTemplateOutlet="roll5"></ng-container></ng-container>
<ng-container *ngSwitchCase="6"><ng-container *ngTemplateOutlet="roll6"></ng-container></ng-container>
</ng-container>
<ng-template #roll1>Too bad we're not playing cards!</ng-template>
<ng-template #roll2>Bad luck!</ng-template>
<ng-template #roll3>Halfway there!</ng-template>
<ng-template #roll4>Better luck next time!</ng-template>
<ng-template #roll5>Very good!</ng-template>
<ng-template #roll6>Jackpot!</ng-template>
