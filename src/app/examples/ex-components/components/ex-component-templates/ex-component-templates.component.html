Component Templates

String Interpolation

Component to Template

The variable 'title' from the component context is interpolated in the template.

component code
title = 'Hello world!';

component template
{{title}}
output
{{title}}

Property Binding

Component to Template

The variable 'title' from the component context is set as the property value.

component code
title = 'Hello world!';

component template
<input type="text" [value]="title" />
output

Event Binding

Template to Component

A method 'onClick' is added to the component. This method will show an alert dialog. In the template a button is added and the 'onClick' method is bound to the click event.

component code
onClick(): void {
  alert('you clicked');
}

component template
<button (click)="onClick()">Click me<button>
output

Event Binding (with arguments)

Template to Component

Same as previous but we include a parameter.

component code
onClick(text: string): void {
  alert(text);
}

component template
<button (click)="onClick('Hello World')">Click me<button>
output

Two-way-Binding

Component to Template and Template to Component

We can set the [(ngModel)] attribute to a variable in the component. Note: you must import the FormsModule from @angular/forms.

component code
name: string;

clearName(): void {
  this.name = '';
}

component template
<input type="text" [(ngModel)]="name" />
<button (click)="clearName()>Clear<button>
{{name}}
AppModule
import { FormsModule } from '@angular/forms';
imports: [ BrowserModule, FormsModule ]

output


{{name}}
