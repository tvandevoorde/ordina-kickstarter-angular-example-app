DecimalPipe

Transforms a decimal number to a string accorsing to locale rules.

{{ value_expression | number [ : digitsInfo [ : locale ] ] }}

Digitsinfo: minDigits.minFractionDigits-maxFractionDigits

Unformatted
{{ number }}
{{ number }}
Formatted by locale
{{ number | number }}
{{ number | number }}
Formatted by argument
{{ number | number: '3.1-5' }}
{{ number | number: '3.1-5' }}
{{ number | number: '4.5-5' }}
{{ number | number: '4.5-5' }}
Formatted by custom locale
{{ number | number: '3.5-5':'en-UK' }}
{{ number | number: '4.5-5':'en-UK' }}

DatePipe

Transforms a date to a string according to locale settings.

{{ value_expression | date [ : format [ : timezone [ : locale ] ] ] }}

See the angular documentation

Unformatted
{{ date }}
{{ date }}
Formatted by locale
{{ date | date }}
{{ date | date }}
Formatted by custom format
{{ date | date: 'dd-MM-yyyy' }}
{{ date | date: 'dd-MM-yyyy' }}
{{ date | date: 'short':'UTC+9':'jp' }}
{{ date | date: 'short':'UTC+9':'ja' }}

Note: you have to register the locale and locale extra first for this to work!
